Filter System

Time Period
?month=2025-01
All Months Jan–Dec 2025 Q1–Q4 2025
Warehouse / Location
?warehouse=London
All San Francisco London Tokyo
Category
?category=sensors
All Circuit Boards Sensors Actuators Controllers Power Supplies
Order Status
?status=delivered
All Delivered Shipped Processing Backordered
Note: Inventory does not support month filtering — it is a static snapshot with no time dimension. Demand and Backlog views ignore all filters.

API Endpoints

GET /api/inventory warehouse, category
GET /api/inventory/{id} single item
GET /api/orders warehouse, category, status, month
GET /api/orders/{id} single order
GET /api/dashboard/summary all filters · KPI totals
GET /api/demand no filters
GET /api/backlog no filters
GET /api/spending/summary overall cost totals
GET /api/spending/monthly monthly breakdown
GET /api/spending/categories by category
GET /api/spending/transactions transaction history
GET /api/reports/quarterly quarterly performance
GET /api/reports/monthly-trends month-over-month trends
